Detailed Description

Window image class defines a accessable image.

This class defines an accessable window image. Each window image is constructed with 8 given uint8_t hex codes. The font of an can be accessed with the operator[]. This will return a uint8_t value that corresponds with the given position This class is inspired by the hwlib's image class. Link to file: https://github.com/wovo/hwlib/blob/master/library/graphics/hwlib-graphics-image.hpp
